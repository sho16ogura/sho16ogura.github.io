Breakdown of the Website and its Workings:

  • Front/ Home Page: A landing page with a navigation bar and a footer. It displays a welcome message and contains a link to get started with the CSV reader.
  • Upload Page: The home page directs to this page which allows the user to input a csv file or a folder with multiple files to read. After uploading is successful the page leads to the display option pages.
  • Display options Pages: This page shows various options for the user to choose how to read certatin datas from the csv they uploaded. It includes a Bar Graph, Pass and Fail, Analyzing etc.
  • Instruction Page: This page will hold instructions or guidelines for the user on how to use the website and its purpose.
  • Documentation Page: This is mostly for the team and Professor Gerten to look through to understand the workings and functionality of the project.
  • About the Team Page: A brief introduction to the Team and the project.
